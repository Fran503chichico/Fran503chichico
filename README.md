## 📁 Proyectos Destacados 🚀

### 📱 Mobile Book & Task App (`branch: master`)
* **Core:** Desarrollo móvil nativo modular utilizando la carpeta raíz `app/` para un ruteo limpio y escalable.
* **Feature Clave:** Integración de persistencia de datos local con `AsyncStorage`, renderizado fluido de interfaces basadas en prototipos de Figma e historial dinámico.
* **Tech Stack:** `React Native` | `Expo` | `JavaScript`

### 🌐 Dynamic Inventory & Order Dashboard
* **Core:** Sistema web Full-Stack con arquitectura optimizada para la gestión de registros masivos en tiempo real.
* **Feature Clave:** Implementación de sincronización asíncrona mediante AJAX/jQuery para operaciones CRUD instantáneas sin refrescar el navegador.
* **Tech Stack:** `PHP` | `MySQL` | `jQuery` | `AJAX`
